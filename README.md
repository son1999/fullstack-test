### Installing Server

Move in server folder run command

```
cd server
```

And run

```
cp .env.example .env
```

Next

```
npm install
```

Next

```
npm run start:dev
```

### Installing Client

Move in server folder run command

```
cd client
```

And run

```
cp .env.example .env.local
```

Next

```
npm install
```

Next

```
npm run dev
```
